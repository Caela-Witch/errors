Caela's HTTP Error Generator

Welcome to the Caela's HTTP Error Generator API!

Usage

Generate an error: Send a GET, POST, PUT, DELETE, or PATCH request to /error with a code parameter.

Example Requests:

GET Request:

curl "https://errors.caela.me/error?code=404"

POST Request:

curl -X POST https://errors.caela.me/error -H "Content-Type: application/json" -d '{"code": 500}'

Enjoy testing!
